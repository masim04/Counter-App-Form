# React Counter App (with Form) + Tailwind

A small React app that increments/decrements a counter and updates the counter using a form input. Styled with Tailwind CSS and organized into simple components.

## Installation
npm install  
npm run dev

## Usage
- Click + or - to change the counter
- Enter a number in the form and submit to set the counter

## Files
App.jsx  
CounterControls.jsx  
CounterForm.jsx  
DisplayCounter.jsx

## Tech
React + Tailwind CSS

## Build
npm run build
